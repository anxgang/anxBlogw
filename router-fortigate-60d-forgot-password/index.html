    [Router] Fortigate 60D 忘記密碼處理

    2016-12-10 少於 1 分鐘閱讀

     索引

    • 前言
    • 重置密碼方式
    • 小小的心路歷程

    前言

    Fortigate 60D 忘記密碼處理方式有二

    1. 重置密碼
      • 用console連接電腦的com port，再用terminal連 - 用usb連接電腦，再用fortiExplorer連
    2. Reset全部(當然是下下策啦，但平常備份有做好做滿的話，也還ＯＫ)

    重置密碼方式

    ##1. 預備工作

    1. 安裝fortiExplorer並開啟程式
    2. 接上 usb ( Router後方mini usb接電腦usb)
    3. 這時候可以看到連接上裝置(當然Router要通電）
    4. 點開CLI介面可以看到裝置的SN，先複製下來備用

    ##2. 心理準備

    1. 登入時必須打上帳號及密碼
      • 帳號: maintainer
      • 密碼: bcpbXXXXXXXXXXXXX (XXXXXXXXXXXXX是剛剛複製的SN)
    2. 登入的時間必須要在15秒內完成
    3. 請先把帳號密碼放在一旁準備複製貼上

    ##3. 開始登入

    1. 把插頭拔掉，再插上
    2. 在等他開機的時間，請先把fortiExplorer開好
    3. 等到偵測到裝置，馬上切到CLI頁面
    4. 看到login:出現就表示可以開始登入了
    5. 登入 (請在15秒內完成，失敗需重新再來)
      • 成功會出現 success
      • 失敗會出現 Login incorrect

    ##4. 修改密碼

    1. 請按照下方指令輸入，並把 YOUR_NEW_PASSWORD 改成你要的密碼 
      #config system admin
#edit admin
#set password YOUR_NEW_PASSWORD
#end

    ##5. 登入測試

    1. 這時候可以用fortiExplorer內附的登入頁測試
    2. 或直接連ip測試

    小小的心路歷程

    網路上大部分解法都是”console連com port“，都什麼年代手邊筆電哪有這種port，居然google到的都是這種文章 看著官方的pdf檔，原來用後面的usb連也可以，不一定要用console 另外還有一個fortiExplorer可以用，就不用開其他termainal 但是官方pdf卻沒有寫怎麼登入，還是參考前人的文章，混搭著成功了
